---
title: Canonical Hosts
description: Authoritative hostnames for the Riven AI platform.
---

The Riven AI platform exposes the following canonical hostnames. Please use
these in any documentation, integration, or client code.

## Public (Stack A)
- `chat.rivenai.io` — consumer chat
- `api.rivenai.io` — API root
- `auth.rivenai.io` — OAuth/login
- `docs.rivenai.io` — documentation (this site)
- `status.rivenai.io` — status
- `cdn.rivenai.io` — static assets
- `security.rivenai.io` — disclosure (security.txt)

## Operator (Stack B)
- `console.rivenai.io`
- `console-sbx.rivenai.io`

## Relay (Stack R)
- `relay.rivenai.io`
- `download.rivenai.io`

## Legacy aliases
These will be retired after a 90-day soak with green redirect metrics:
- `workspace.rivenai.io` → `console.rivenai.io`
- `sbx-workspace.rivenai.io` → `console-sbx.rivenai.io`
- `relay-dl.rivenai.io` → `download.rivenai.io`
- `rivenai.com` → `rivenai.io`
